## Installation

Git clone
```
git clone https://github.com/netis-cms/netis.git
```

Install composer
```
composer i
```

## npm
```
npm i
```

## docker build
```
docker-compose build
```

## docker dev up
```
docker-compose -f docker-compose.yml -f docker-compose.dev.yml up
```
